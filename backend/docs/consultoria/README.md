# Documentos de consultoría para el asesor de gerencia

Material de revenue management y gestión hotelera para alimentar el **RAG del asesor del dueño**
(colección `management_knowledge`, separada del conocimiento del agente de huéspedes).

## Cómo cargarlos

Subir cada `.md` desde el backoffice: **Asesoría → subir documento** (acepta PDF, Markdown y
texto). Una vez cargados y activos, el asesor los usa vía la tool `consultar_conocimiento`
para fundamentar y CITAR sus recomendaciones (no responder de memoria).

## Documentos

1. `01-kpis-revenue-management.md` — RevPAR, ADR, ocupación, TRevPAR, GOPPAR, RevPASH.
2. `02-pricing-dinamico.md` — pricing dinámico, estacionalidad, length-of-stay, value-based.
3. `03-forecasting-y-booking-pace.md` — forecasting de demanda, booking pace y pickup.
4. `04-distribucion-y-canales.md` — OTAs vs. directa, comisiones, mix de canales.
5. `05-experiencia-reputacion-y-upselling.md` — reputación, experiencia, ancillary revenue.

Cada documento incluye fuentes citadas (referencias del sector) al pie. Son material genérico
de buenas prácticas; el asesor los cruza con los **datos reales del hotel** (que salen de sus
otras herramientas), no los reemplaza.
